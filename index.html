Welcome to My Portfolio

Project 1: Fruity Frank (JS(p5.js), HTML, CSS) For the second project in the first year of HBO-ICT, the goal was to recreate a classic arcade game, but with a clear twist. Fruity Frank is a spinoff the classic Mario/Donkey Kong game, but with a twist. Instead of having to walk up the platforms to save the princess, you are on top of the platforms and have to throw fruits at incoming enemies to protect your fruit bowl. Fruity Frank Project 2: Last Stand (Unity, C#, PHP) This was my first project in Unity. The goal of the project was to create a game in Unity that is playable in the browser. We were not allowed to use the inspector to change values, so everything had to be done in code. Last Stand is a bullet hell game where you have to survive as long as possible against incoming projectiles. This is a game just using primitive shapes for the enemies and the player. Last Stand Project 3: The Cube (C++, HTML, CSS, JS) In the last semester of my second year, I followed a minor called Internet of Things. I have learned all about microcontrollers, sensors, and how to program them. The Cube was a project for the individual assignment of the minor. It's a small handheld game device that closely resembles "Bop-It!". The Cube Project 4: A-Moebe (Python, HTML, CSS, JS, Flask) A-Moebe is a project that was not made for school, but for a friend that had started his business in building escaperooms. This is an all in one system that can be used to control the escaperoom. It has a timer, a hint system, a way to control the lights and sounds in the room, turn magnetic locks on and off, read sensor-data in realtime and this all dynamically.
